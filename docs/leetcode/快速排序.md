### 快速排序与快速选择算法

在以上两个算法中，一个十分重要的步骤是划分，有两种划分方法

```cpp
void quicksort(vector<int> &nums, int left, int right) {
    if (right - left <= 1) {
        return;
    }
    int i = left;
    int j = right;
    while (i < j) {
        do ++i; while (i < right && nums[i] < nums[left]);
        do --j; while (nums[j] > nums[left]);
        if (i < j) {
            swap(nums[i], nums[j]);
        }
    }
    swap(nums[left], nums[j]);
    quicksort(nums, left, j);
    quicksort(nums, j + 1, right);
}
```

```cpp
void quicksort(vector<int> &nums, int left, int right) {
    if (right - left <= 1) {
        return;
    }
    int slow = left + 1;
    int fast = left + 1;
    while (fast < right) {
        if (nums[fast] <= nums[left]) {
            swap(nums[slow], nums[fast]);
            ++slow;
        }
        ++fast;
    }
    swap(nums[left], nums[slow - 1]);
    quicksort(nums, left, slow - 1);
    quicksort(nums, slow, right);
}
```

通常采用**第一种**方法，因为在面对数组中有多个相同值的情况时，
第一种方法对数组的划分更均匀，运行速度更快。第二种方法很可能超时
