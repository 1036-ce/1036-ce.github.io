## 快速幂算法

```cpp
// quickMul(x, n) == pow(x, n)
double quickMul(double x, int n) {
    double ret = 1.0;
    double num = x;
    while (n > 0) {
        if (n & 1) {
            ret *= num;
        }
        num *= num;
        n >>= 1;
    }
    return ret;
}
```

## 举例解释

对于pow(x, 6)可以分解为pow(x, 4) * pow(x, 2), 4和2分别对应于6的二进制表示中bit为1的部分
